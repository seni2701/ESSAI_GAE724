# Contribuer à ESSAI_GAE724

## Signaler un problème
Ouvrez une **Issue** avec :
- Description précise du problème
- Environnement (OS, Python, conda env)
- Message d'erreur complet + cellule concernée

## Proposer une amélioration
1. Forkez le dépôt
2. Créez une branche : `git checkout -b feature/nom-amelioration`
3. Commitez : `git commit -m "Description claire"`
4. Ouvrez une **Pull Request** vers `principal`

## Standards
- PEP 8 pour le style Python
- Commentaires en français
- Docstrings pour toutes les fonctions GEE et ML
- Tester sur crop_env (Python 3.10) avant PR
